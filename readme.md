# 🎮 Projekt: Hra inspirovaná hrou Polda

## 📌 Úvod
Tento projekt je webová hra vyvíjená pomocí moderních technologií **Next.js** a **React**.  
Hra je inspirována českou adventurou *Polda*, která je známá svým příběhem, dialogy a interakcí s prostředím.

Cílem projektu není vytvořit kopii původní hry, ale převzít její základní principy a aplikovat je v rámci webové aplikace.

---

## 🎯 Cíl projektu
- vytvořit jednoduchou adventurní hru běžící v prohlížeči  
- implementovat interakci s herním prostředím  
- pracovat s dialogy a herní logikou  
- využít React komponenty pro strukturování aplikace  
- pochopit fungování frameworku Next.js  

---

## 🛠 Použité technologie

- **Next.js** – framework pro React (routing, server-side rendering)
- **React** – komponentový systém pro tvorbu UI
- **JavaScript / TypeScript** – logika aplikace
- **CSS / Tailwind (volitelně)** – stylování

---

## 🎮 Princip hry

Hra funguje na podobném principu jako klasické point-and-click adventury:

- hráč ovládá postavu
- může klikat na objekty v prostředí
- spouští dialogy nebo akce
- postupně plní jednoduché úkoly

---

## 🧩 Funkcionalita (aktuální / plánovaná)

- pohyb mezi scénami
- interakce s objekty
- dialogový systém
- inventář (plánováno)
- ukládání postupu (plánováno)

---

## 📁 Struktura projektu

- `app/` – stránky a routing (Next.js)
- `components/` – znovupoužitelné React komponenty
- `styles/` – Tailwind
- `public/` – obrázky a statické soubory
